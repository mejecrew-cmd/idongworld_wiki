# 아이동월드 위키 (IDONG World Wiki)

아이돌과 동물이 만난 작은 행성, **아이동월드**의 세계관·도감 정적 위키.

## 구성
- **위키 홈** — `index.html`
- **아이템 카드북** — `item-card/` · 2,832종 아이템 도감
  - 분류(11) · 태그(소장/기능/소재/형태) · 아이동(671) 다축 탐색
  - 검색 · 정렬 · 단독 상세 페이지(사물명 · IPA · 설명 · 수집 아이동)
  - 모바일 한 줄 리스트 · 접이식 태그 구름
- **아이동 월드북** / **아이동 시트북** — 준비중

## 기술
순수 정적 HTML/CSS/JS. 데이터는 `item-card/data/*.js`(`window.IDW_*`)로 분리되어
빌드·서버 없이 `file://` 로도 열람 가능합니다.

번역 작업용 문자열 정본: `item-card/아이템_스트링_데이터.csv`
